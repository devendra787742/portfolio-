# 🚀 Devendra Singh Portfolio Website

A modern, interactive, and visually rich personal portfolio website built using pure HTML, CSS, and JavaScript. This project showcases my skills, projects, experience, and achievements in a dynamic and engaging way.

---



---

## 📌 Features

✨ **Modern UI/UX Design**

* Clean and futuristic interface with neon-inspired theme
* Smooth animations and transitions

🎮 **Interactive Background (Game-like Experience)**

* Animated stars, mountains, clouds, and cityscape
* Parallax effects based on mouse movement
* Particle effects on cursor movement and clicks

🖱️ **Custom Cursor**

* Unique animated cursor with hover effects

📱 **Responsive Design**

* Fully responsive for mobile, tablet, and desktop

📊 **Portfolio Sections**

* About Me
* Professional Summary
* Experience
* Projects
* Skills
* Certifications
* Education
* Contact

---

## 🛠️ Technologies Used

* **HTML5** – Structure of the website
* **CSS3** – Styling, animations, and layout
* **JavaScript (Vanilla JS)** – Interactivity and dynamic effects
